# TCP Incredible Dream Map

Revenue Journey Architecture for The Concrete Protector dealer network. A visual board-game-style map that tracks the full customer lifecycle from discovery marketing through client growth, training, and community membership.

## Overview

The Dream Map visualizes TCP's complete revenue process across multiple "deals" (stages):

- **Deal 1 (FIND):** Discovery marketing funnel - Stranger to Customer ($499 deposit)
- **Deal 2 (Training Class):** In-person or remote certification - Customer to L1 Certified
- **Deal 3 (C2C Success):** Customer-to-Client journey - Strategy, GTM, Agreement, Launch
- **Client Growth (Shield Path):** L1-L7 progression with revenue milestones
- **ICC:** Incredible Collaborative Community - Member, Ambassador, Consultant
- **Delight:** Customer service path for product-only customers

## Tech Stack

- HTML5 / CSS3 (single-file, self-contained)
- Vanilla JavaScript (tab switching)
- Google Fonts (Playfair Display, DM Sans, JetBrains Mono)
